只做文件配置，不做 rclone 安装或网盘授权（OAuth 授权无法自动化，需要 SSH 进服务器手动跑 rclone config）。保存后需要重启 aria2 才生效：docker 模式 docker compose restart aria2， 裸机模式 systemctl restart tg-aria2-bot-aria2。

GoFile 上传

下载完成后（可选）压缩、上传到 gofile.io、 再（可选）删除本地文件，全部在机器人进程内完成，不依赖 rclone。不填 token 是匿名上传，文件会过期； 去 gofile.io 个人资料里拿 token 填这里可以让文件归到你账号名下。保存后需要重启机器人才生效： docker 模式 docker compose restart bot，裸机模式 systemctl restart tg-aria2-bot。
